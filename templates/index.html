Описание курса

На английском Python звучит как «пайтон», но среди программистов более распространено название «питон». А ведь происхождение слова не связано с пресмыкающимся. Автор языка Гвидо ван Россум назвал его в честь комедийного телешоу «Летающий цирк Монти Пайтона» (Monty Python’s Flying Circus).

Для изучения Python не требуется специальных знаний. Достаточно владеть компьютером на уровне пользователя. Благодаря своим неоспоримым достоинствам язык широко востребован на рынке. Среди компаний, которые его используют: Google, Dropbox, Yandex, Intel, Cisco, Hewlett-Packard, IBM. На Python разрабатывались Ubuntu, BitTorrent и YouTube.

Длительность курса: 36 ч.
