Description of PyBursa course

Чем интересен наш курс?

Практический курс по изучению современного языка программирования Python и фреимворка Django. Обучение будет проходить с уклоном в веб-разработку. Цель PyBursa научить начинающих программистов до уровня Junior Python Developer и после окончания курса все ученики могут претендовать на эту должность в любой интернет-компанию. Весь курс построен на практике: преподаватели-практики, обучение на примере разработки реальных проектов, практические домашние задания. На курсе мы Вас познакомим с языком Python, фреимворком Django, научим работать с базами данных, покажем, как проводить UnitTest, вместе сделаем работающий проект на Python / Django и освоим еще много полезного. После успешного окончания обучения всем предоставляется именной сертификат, который высылается почтой владельцу. Для нас это уже третья группа, и многие из наших учеников успешно работают в сфере Python Development. Курс совершенствуется из года в год и становится еще более эффективным.


Program of PyBursa course

  1. Основы Python
    • zen, pep8, pylint, tools, IDE
    • консоль, типы данных, операторы
    • синтаксис управляющих конструкций (if, for...)
    • исключения и разбор стектрейса
    • кодировка в Python
  2. Строки и итераторы
    • работа со строками и итерируемыми типами
    • запуск скрипта и параметры
    • git http://try.github.com/levels/
  3. Функции и модульность
    • функции (def, lambda)
    • генераторы списков
    • модули, структура
  4. Библиотеки и пакетный менеджер
    • встроенные библиотеки http://docs.python.org/
    • сторонние библиотеки (менеджер пакетов) PYPI
    • PYTHONPATH и виртуальное окружение
  5. Классы и ООП, основы
    • синтаксис, атрибуты, методы, инстанцирование
    • все объект, служебные методы
  6. Классы и ООП, возможности Python
    • инкапсуляция, наследование, полиморфизм
    • переопределение методов
    • super
    • абстрактные классы
    • статические методы
  7. Деплоймент
    • nginx
    • Управление статикой
    • Деплоймент
